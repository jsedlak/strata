# Strata

![Strata Logo](img/strata-logo.png)

Strata is an opinionated Event Sourcing library built for Microsoft Orleans.

## Features

- ♾️ Tentative & Confirmed State Models & Versioning
- 📷 Snapshotting
- ⏱️ Delayed Writes
- 🗃️ Orleans Provider Model
